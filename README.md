# Слайдер
Слайдер для сайта, показывающий завершённые проекты. Реализован на чистом JavaScript.

## Особенности
Переключение стрелками, точками и текстовыми ссылками

Кольцевая навигация

Управление клавишами ← и →

Адаптивный дизайн

## Быстрый старт
Скопируйте файлы: index.html, style.css, script.js

Создайте папку img и добавьте изображения:

Rostov-on-Don, Admiral.png

Sochi_Thieves.png

Rostov-on-Don_Patriotic.png

left_arrow.svg

right_arrow.svg

Откройте index.html в браузере

## Технологии
HTML5

CSS3 (Flexbox)

Vanilla JavaScript

# Слайдер

Слайдер для сайта, показывающий завершённые проекты. Реализован на чистом JavaScript.

## Особенности

- Переключение стрелками, точками и текстовыми ссылками
- Кольцевая навигация
- Управление клавишами ← и →
- Адаптивный дизайн

## Быстрый старт

1. Скачайте проект
2. Сохраните файлы в удобное место
3. Откройте index.html в браузере

## Технологии
- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
